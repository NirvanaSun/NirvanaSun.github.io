python:用polyfit()拟合曲线

发表于 2018-09-11 | 分类于 编程算法技术

numpy中提供了一个可以进行多项式拟合的函数，polyfit(),官网文件详见https://docs.scipy.org/doc/numpy/reference/generated/numpy.polyfit.html#numpy.polyfit
最小二乘多项式拟合。

拟合多项式p（x）＝p〔0〕*x**DEG+…+DEG的程度DEG到点（x，y）。返回最小平方误差的系数p的向量
polyfit(x,y deg,)
其中，x, y 分别是数据点的横坐标和纵坐标，deg是拟合多项式的最高次幂
例如，
import numpy as np
import matplotlib.pyplot as plt
x=np.linspace(1,15,20)
print(x)

y=np.random.random(10)#生成一个随即矩阵

pv=np.polyfit(x,y,8)#拟合成5阶多项式
pv1=np.ploy1d(pv)#曲线的表达式
